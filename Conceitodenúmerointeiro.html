Conceito de número inteiro

No 7.º ano, os números inteiros mostram como os naturais são ampliados para incluir o zero e valores negativos. Este conteúdo é importante para entender comparações, operações e o uso de sinais.

O que são os números inteiros?

O conjunto dos números inteiros é denotado por ℤ e inclui:

  • Inteiros positivos: 1, 2, 3, ...
  • Zero: 0
  • Inteiros negativos: -1, -2, -3, ...

Por que estudar inteiros?

  • Comparar e ordenar valores: em inteiros, -10 é menor que -2.
  • Somar e subtrair: 3 - 5 pode ser escrito como 3 + (-5) = -2.
  • Multiplicação e Divisão (Regra dos Sinais): Dominar como os sinais interagem.
    • Sinais iguais dão resultado positivo (ex: -3 * -4 = +12).
    • Sinais diferentes dão resultado negativo (ex: -12 \ 3 = -4\).
  • Resolução de Problemas: Aplicar estas regras a contextos reais e expressões numéricas.

Regra dos sinais

  • Sinais iguais: resultado positivo. Ex: -3 × -4 = +12.
  • Sinais diferentes: resultado negativo. Ex: -12 ÷ 3 = -4.

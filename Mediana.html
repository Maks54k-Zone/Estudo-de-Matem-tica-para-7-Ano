Mediana

A mediana é o valor central de um conjunto de dados ordenados. Para conjuntos com um número par de valores, a mediana é a média dos dois valores centrais.

Como encontrar a mediana

  • Ordene os valores do menor para o maior.
  • Se houver um único valor central (n impar), esse é a mediana.
  • Se houver dois valores centrais (n par), a mediana é a média aritmética desses dois valores.
Exemplo:
Números: 5, 3, 9, 11, 15 → Ordenados: 3, 5, 9, 11, 15 → Mediana: 9

Observação: média aritmética

Alguns podem perguntar: «O que é a média aritmética?» A média aritmética é o valor obtido ao somar todos os dados e dividir pelo número de dados.

Exemplo: para o conjunto 15, 4, 2, 17, 13, 3 — ordenando: 2, 3, 4, 13, 15, 17. Como o conjunto tem um número par de elementos, a mediana é a média dos dois valores centrais (4 e 13): (4 + 13) / 2 = 8.5 que agora já é a mediana.
